📋 Instruções de Uso ×

Como configurar sua planilha do Google Sheets:

  1. Torne a planilha pública: Vá em "Arquivo" > "Compartilhar" > "Publicar na Web". Certifique-se de que está publicada. Depois, em "Compartilhar" (botão verde/azul), mude a opção "Restrito" para "Qualquer pessoa com o link" e defina o papel como "Leitor". Isso é crucial para a API de visualização funcionar.
  2. Obtenha o ID da planilha: Copie da URL. Se a URL for `https://docs.google.com/spreadsheets/d/1fSTEyMmAEgfMnAFUTdYbJah5dr8OudsEsqjKHnHwFms/edit#gid=1536647709`, o ID é `1fSTEyMmAEgfMnAFUTdYbJah5dr8OudsEsqjKHnHwFms`.
  3. GID da aba (opcional, mas recomendado): Para abas específicas, use o número após `gid=` na URL. No exemplo acima, o GID é `1536647709`. Se não for fornecido, tentará carregar a primeira aba visível.

Estrutura esperada da planilha (primeira linha é cabeçalho, ignorado):

  • Coluna A: Tipo de Processo (pode conter múltiplos separados por vírgula, ex: "VIDA, AUXILIO-ACIDENTE")
  • Coluna B: Nome do Cliente
  • Coluna C: Link (opcional, não utilizado pelo dashboard)
  • Coluna D: Data (formato: DD/MM/AAAA)
  • Coluna E: Nome do Consultor

Importante: A primeira linha da sua aba deve ser o cabeçalho (ex: TIPO, CLIENTE, LINK, DATA, CONSULTOR). Os dados reais devem começar da segunda linha.

Recursos do Dashboard:

  • Conexão Real: Busca dados diretamente da sua Google Sheet.
  • Contagem múltipla: Cada processo na mesma linha é contabilizado individualmente.
  • Filtro por mês: Selecione o mês para visualizar as vendas.
  • Auto-update: Atualiza automaticamente a cada 30 segundos (pode ser pausado).
  • Fallback: Caso não consiga conectar após várias tentativas, mostra dados demonstrativos.
  • Logs detalhados: Registra as operações de conexão e processamento.
